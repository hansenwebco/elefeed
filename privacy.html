Privacy Policy

This Privacy Policy outlines the data processing practices of Elefeed ("the Application").

1. Information Collection and Processing

The Application does not collect, process, transmit, or retain any personally identifiable information, usage metrics, diagnostic data, or activity logs on external network infrastructure or proprietary servers.

2. Third-Party Analytics and Tracking

The Application strictly prohibits the use of third-party analytics services, external tracking mechanisms, embedded scripts, or tracking cookies. Continuous or aggregated user behavior monitoring is not performed.

3. Local Data Storage and Synchronization

All operational data required for the Application's functionality, including but not limited to authentication credentials (such as OAuth access tokens) and user-defined application preferences, are maintained primarily within the Local Storage API of the user's web browser environment. If the optional Daily Usage or Settings Sync features are enabled, certain preferences and usage timers may be synced to your Mastodon account's private Relationship Note field—a secure, personal field visible only to you. This data is utilized solely for the purpose of synchronizing settings across devices and providing usage statistics as requested. No application data leaves the client device to interact with servers affiliated with the Application developers.
